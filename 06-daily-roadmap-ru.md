# Marketplace Seller Simulator — поминутный план по дням (Яндекс Игры)

Цель: **играбельный M1 за ~3 недели**, затем **сборка, готовая к релизу на Яндекс Играх**, по фазам из [05-early-access-roadmap-ru.md](05-early-access-roadmap-ru.md).

**Как читать:** один **день** = один сфокусированный рабочий день (ориентир **4–6 часов**). Меньше времени — умножай сроки. Семь дней в неделю — можно быстрее.

---

## Фаза 0 — подготовка (до дня 1, 0.5–1 день)

- Зафиксировать стек: браузерный JS, JSON-данные, опциональный бандлер.
- Репозиторий, `index.html`, `app.js`, команда dev-сборки.
- Константы из [01-economy-v1-ru.md](01-economy-v1-ru.md) и M1 из [04-m1-prototype-scope-ru.md](04-m1-prototype-scope-ru.md) в `config.js`.
- Заготовка модуля Яндекс SDK: `src/platform/yandexSdk.js`.

---

## M1 — прототип (дни 1–21)

Обязательный объём — [04-m1-prototype-scope-ru.md](04-m1-prototype-scope-ru.md). **С 10-го дня разработки новых систем не добавлять** — только полировка, баги, баланс, UX.

| День | Задача |
|------|--------|
| **1** | Каркас: `src/core`, `data`, `ui`, `persistence`, `platform`. Базовое состояние игры. |
| **2** | `categories.json` + `skus_beauty_m1.json`, 10 SKU, загрузка. |
| **3** | Кнопка «следующий день», жизненный цикл рана (`startRun`, `endRun`). |
| **4** | `demandModel.js`: органика + платный трафик. |
| **5** | Конверсия и заказы с потолком по `inStock`. |
| **6** | Закупка: очередь входящих партий. |
| **7** | Приход партий в начале дня, обновление остатков. |
| **8** | Карточка: пресет качества, цена, промо. |
| **9** | Реклама: дневной бюджет, списание в cost model. |
| **10** | **Заморозка объёма.** Модель затрат: комиссия, логистика, оверхед, COGS. |
| **11** | Возвраты и чистая выручка. |
| **12** | Учёт stockout и защита от тупиков. |
| **13** | Дашборд KPI. |
| **14** | Панель разбивки расходов. |
| **15** | Save/load: `localStorage` + экспорт JSON. |
| **16** | Обёртка `ysdk`, fallback без SDK. |
| **17** | Rewarded в конце рана (опциональная награда). |
| **18** | Interstitial после итогов с кулдауном. |
| **19** | Плейтест №1, топ-3 UX и экономика. |
| **20** | Плейтесты №2–3, телеметрия включая рекламу. |
| **21** | Freeze M1, черновик чеклиста ассетов для витрины. |

Опционально при запасе времени: 3 события из [03-content-v1-ru.md](03-content-v1-ru.md), мини-ветка на 4 узла, итог недели.

## Привычки на 15 минут в день

- Одна строка в журнале: сделано / завтра.
- Один стабильный билд или git-tag в неделю.
- Короткий прогон 7 игровых дней после смены экономики.
